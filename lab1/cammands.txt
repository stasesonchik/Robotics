===
Название: head
Описание: выводит первые строки файла
Синтаксис: head [опции] имя_файла
Пример: head -n 5 file.txt
Примечание: удобно просматривать начало больших файлов

===
Название: tail
Описание: выводит последние строки файла
Синтаксис: tail [опции] имя_файла
Пример: tail -n 5 file.txt
Примечание: с опцией -f можно следить за логами в реальном времени

===
Название: grep
Описание: ищет строки в файле по шаблону
Синтаксис: grep [опции] "шаблон" файл
Пример: grep "error" logfile.txt
Примечание: очень полезно для поиска по тексту и логам

===
Название: find
Описание: ищет файлы и каталоги по критериям
Синтаксис: find [каталог] [условия]
Пример: find /home/user -name "*.txt"
Примечание: позволяет искать файлы по имени, размеру, дате и др.

===
Название: chmod
Описание: изменяет права доступа к файлам и каталогам
Синтаксис: chmod [опции] права файл
Пример: chmod 644 file.txt
Примечание: 644 означает чтение и запись для владельца, чтение для остальных

===
Название: chown
Описание: изменяет владельца и группу файла или каталога
Синтаксис: chown [опции] владелец[:группа] файл
Пример: chown user:user file.txt
Примечание: часто используется для настройки прав в системе

===
Название: df
Описание: показывает использование диска для файловых систем
Синтаксис: df [опции]
Пример: df -h
Примечание: -h делает вывод в удобных единицах (KB, MB, GB)

===
Название: du
Описание: показывает размер файлов и каталогов
Синтаксис: du [опции] файл/каталог
Пример: du -sh /home/user
Примечание: -s даёт общий размер, -h выводит в удобных единицах

===
Название: sort
Описание: сортирует строки в файле или потоке
Синтаксис: sort [опции] файл
Пример: sort data.txt
Примечание: можно сортировать по числам, алфавиту, менять порядок сортировки

===
Название: uniq
Описание: удаляет повторяющиеся строки или показывает уникальные
Синтаксис: uniq [опции] файл
Пример: sort data.txt | uniq
Примечание: часто используется вместе с sort для фильтрации дубликатов
